# Omarchy

> A new Linux distribution project led by [[dhh]], designed to simplify the desktop Linux experience.

## Overview

Omarchy is an example of an ambitious project enabled by the "AI-pilled" workflow. [[dhh]] describes using AI agents to handle the massive implementation burden of building and packaging a Linux distribution, allowing him to focus on the design, "vibes," and user experience of the OS.

## Significance

It serves as a case study for the [[peak-programmer]] thesis: an individual or tiny team taking on a project that would traditionally require a large engineering organization.
