# Qinggang Zhang

> Qinggang Zhang is a researcher at Jilin University and corresponding author of the MemGraphRAG paper (KDD 2026). His work focuses on GraphRAG, multi-agent systems, and memory-augmented retrieval.

## Affiliation
- Jilin University, Changchun, China
- Email: qinggangzhang@jlu.edu.cn

## Key Contributions
- **MemGraphRAG** (KDD 2026): Memory-based multi-agent system for GraphRAG with Three-Layer Global Memory and Personalized PageRank retrieval
- **HippoRAG / HippoRAG2** (with collaborators): Associative memory approaches for GraphRAG using Personalized PageRank
- **FaithfulRAG** (arXiv 2025): Fact-level conflict modeling for context-faithful RAG
