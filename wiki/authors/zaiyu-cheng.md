# Zaiyu Cheng

> Researcher at William & Mary; lead author of the 360-configuration empirical study on system prompt effects in code generation.

Conducted the first large-scale controlled study examining how system-level instructions influence instruction-tuned models in code generation tasks, spanning 360 configurations across 4 models, 5 system prompts, 3 prompting strategies, 2 languages, and 2 temperature settings. Key finding: system prompt specificity does not monotonically improve correctness.
