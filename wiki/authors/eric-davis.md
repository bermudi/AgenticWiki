# Eric Davis

> Researcher at Galois, Inc. (Portland, OR) working on human-machine teaming and biologically-inspired AI memory. First author of the episodic + semantic memory architecture for machine teammates.

## Contributions

- **Episodic + semantic memory for machine teammates** ([[episodic-memory-for-agents]]). With [[katrina-schleisman]]: integrates episodic memory (spatiotemporal events) with a self-organizing-map-based semantic memory, and introduces **meta-episodes** — co-training interventions represented as a labeled transition system over semantic clusters. The explicit target is **co-learning and co-training**: enabling an AI teammate to participate in after-action review and set goals for its own skill development. Evaluated on helicopter air ambulance (HAA) trajectory data.
