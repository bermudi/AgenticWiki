# Xuying Ning

> Lead author and core contributor of the *Code as Agent Harness* survey (arXiv:2605.18747), formally defining the framework of code as the executable, inspectable, and stateful substrate for AI agent systems.

## Affiliation

University of Illinois Urbana-Champaign (UIUC)

## Contributions

- Lead author of the first comprehensive survey formalizing **code as agent harness** — the view that code is not just an artifact LLMs generate, but the operational medium through which agents reason, act, and coordinate.
- Organized the literature (up to 2026) into three connected layers: harness interface (code for reasoning, acting, environment modeling), harness mechanisms (planning, memory, tool use, control, optimization), and scaling the harness (multi-agent orchestration over shared code artifacts).
- Identified seven open problems for harness engineering: harness-level evaluation, semantic verification, self-evolving harnesses, transactional shared state, HITL safety, multimodal systems, and the broader science of harness engineering.

## Co-authors (Core Team)

The survey's core contributors span UIUC, Meta, and Stanford, including Katherine Tieu, Tianxin Wei, Zihao Li, Yuanchen Bei (UIUC); Dongqi Fu, Zikun Cui, Yang Cao, Pan Chen, Dorothy Sun, Ren Chen, Mahesh Srinivasan, Nipun Mathur, Yinglong Xia, Hong Li, Hong Yan (Meta); Pan Lu, Jiaru Zou (Stanford); and senior advisors Lingming Zhang, Tong Zhang, Hanghang Tong, and Jingrui He (UIUC).
