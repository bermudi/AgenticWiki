# Mathis Pink

> Researcher at the Max Planck Institute for Software Systems. Lead author of the position paper arguing episodic memory is the missing piece for long-term LLM agents.

## Contributions

- **Episodic memory for long-term LLM agents** ([[episodic-memory-for-agents]]). Lead author (with Wu, Vo, Turek, Mu, Huth, Toneva) of the position paper operationalizing episodic memory for LLM agents via five properties (long-term, explicit, single-shot, instance-specific, contextualized), showing no current memory method satisfies all five, and proposing a unifying framework grounded in Complementary Learning Systems theory with a four-direction roadmap (encoding, retrieval, consolidation, benchmarks). Also authored prior work (Pink et al., 2024) on instance-specific temporal-order memory benchmarks.
- Correspondence: `mpink@mpi-sws.org`.
