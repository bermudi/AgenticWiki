# Shubham Ugare

> Researcher at Meta; lead author of the agentic code reasoning paper introducing semi-formal reasoning for execution-free code verification.

Introduced semi-formal reasoning, a structured prompting methodology that requires LLM agents to construct explicit premises, trace execution paths, and derive formal conclusions as verifiable certificates. The approach bridges unstructured chain-of-thought and fully formal verification, achieving 93% patch equivalence accuracy on real-world patches with Opus-4.5. Also demonstrated gains on code question answering (RubberDuckBench, +9pp) and fault localization (Defects4J, +5–12pp).
