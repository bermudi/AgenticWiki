# Lechen Zhang

> Researcher at University of Illinois Urbana-Champaign; lead author of SPRIG, the ICLR 2026 paper on system prompt optimization via genetic algorithms.

First author of SPRIG, which demonstrates that a single optimized system prompt can match per-task prompt optimization across 47 tasks. The work introduces a genetic algorithm over 9,000 prompt components, showing that optimized system prompts converge on CoT and behavioral components while de-prioritizing role components.
