# Dex Horthy

> CTO of [[sourcegraph]] and a key figure in the development of [[cody]].

## Overview

Dex Horthy is a software engineer and leader focused on code intelligence and developer productivity. He advocates for rigorous engineering practices when using AI assistants, coining the term "No Vibes Allowed" to describe a rejection of superficial AI-assisted coding in favor of deep context and verification.

## Philosophy

- **Engineering Rigor**: Believes that LLMs should be used within a structured engineering framework rather than as a "black box" that produces code by "vibing."
- **Code Intelligence**: Emphasizes the importance of symbol-level understanding and graph-based context for AI agents.
