# Sourcegraph

> Universal code search and intelligence platform, and the company behind [[cody]] — an AI assistant that uses symbol-level [[code-intelligence]] to ground LLM responses in actual codebase structure.

## Products

- **Code Search**: Allows developers to search and navigate massive codebases with symbolic understanding.
- **[[cody]]**: An AI coding assistant built on top of Sourcegraph's code intelligence platform.

## Mission

Sourcegraph's mission is to make it so that everyone can code. They focus on solving the "Big Code" problem—the increasing complexity and scale of modern software.
