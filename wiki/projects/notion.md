# Notion

> Workspace platform that Litt positions as infrastructure for shared understanding: collaborative documents, HTML blocks for interactive agent output, and now coding agents embedded in team spaces.

## What It Is

Notion is a collaborative workspace and document platform. In the context of agentic coding, it is the substrate for two of Litt's arguments:

- **Shared understanding**: The team-level equivalent of individual comprehension. Documents that multiple people and agents can read and discuss together are a primitive for collective understanding.
- **HTML as agent output**: Notion's HTML-block feature lets agents embed interactive simulations and figures directly into pages, not just static text.

## Relevance to Agentic Coding

Litt describes Notion as the place where his Explain Diff output lives because it is collaborative by default. Teammates can comment on explainer docs, agents can be present in shared chat threads, and the output is not trapped on a local machine. The coding-agent integrations (Claude, Cursor), launched in 2026, extend this: the team builds and discusses code in the same shared space where the rest of the work lives.
