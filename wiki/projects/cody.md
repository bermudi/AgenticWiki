# Cody

> An AI coding assistant developed by [[sourcegraph]] that uses [[code-intelligence]] to provide context-aware help.

## Key Features

- **Context Fetching**: Uses Sourcegraph's embeddings and symbol graph to find relevant code snippets.
- **Repository-Scale Knowledge**: Can answer questions about entire codebases, not just the currently open file.
- **Explain/Refactor/Write**: Performs standard AI coding tasks but with higher fidelity context.

## Relation to "No Vibes"

Cody is designed to mitigate [[vibes-based-engineering]] by grounding its responses in the actual reality of the codebase (symbols, types, and definitions).
